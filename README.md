# Лабораторная работа №8: Адаптивная верстка с CSS Grid
**Студент:** Назаренко Анна
**Группа:** ИСП-232  
**Дата:** 26.02.26 ## Описание проекта

В данной лабораторной работе изучается технология CSS Grid для создания адаптивных макетов. Grid представляет собой двумерную систему компоновки, позволяющую управлять одновременно строками и столбцами.

## Выполненные задания

### Часть 1. Основы Grid 

В этой части были продемонстрированы базовые свойства Grid:

1. **Создание Grid-контейнера** 
2. **Управление столбцами** 
3. **Управление строками**
4. **Отступы между ячейками** 
5. **Именованные области**
6. **Выравнивание элементов**

### Часть 2. Практические примеры 

Созданы три практических примера использования Grid:

#### Пример 1: Сетка 3x3
- Простая сетка из 9 элементов
- Использование `repeat(3, 1fr)` для равных колонок
- Стилизация элементов с зеленым фоном

#### Пример 2: Макет страницы
- Типичная структура веб-страницы 
- Использование `grid-template-areas` для создания областей
- Фиксированная ширина сайдбара и резиновый основной контент

#### Пример 3: Адаптивная галерея
- Автоматическая подстройка под ширину экрана
- Использование repeat
- Эффекты при наведении (увеличение карточек)
